## 2. VPNs (Virtual Private Networks)

**Where it is used**

- Remote work connections
- Public Wi-Fi security
- Corporate networks

**How cryptography is used**

- Encrypted tunnels protect data traffic
- Secure authentication between user and server

**What it protects**

- Browsing activity
- IP address
- Internal company data

**Real-world examples**

- NordVPN
- Cisco

---

## 3. Password Protection and Authentication

**Where it is used**

- Website and app logins
- Operating systems
- Cloud services

**How cryptography is used**

- Passwords are hashed, not stored in plain text
- Salting protects against pre-computed attacks

**What it protects**

- User accounts
- Credentials during data breaches

**Real-world examples**

- Facebook
- Instagram

---

## 4. End-to-End Encrypted Messaging

**Where it is used**

- Messaging applications
- Voice and video calls

**How cryptography is used**

- Public-key encryption for key exchange
- Symmetric encryption for messages
- Digital signatures for authenticity

**What it protects**

- Messages from hackers
- Messages from service providers

**Real-world examples**

- WhatsApp
- Signal

---

## 5. Digital Payments and Online Banking

**Where it is used**

- Online banking
- Credit/debit card payments
- Mobile payment apps

**How cryptography is used**

- Encryption secures transactions
- Digital signatures verify authorization
- Tokenization protects card details

**What it protects**

- Financial data
- Transaction integrity
- User identity

**Real-world examples**

- Visa
- PayPal

---

## 6. Data Encryption at Rest

**Where it is used**

- Hard drives and SSDs
- Databases
- Cloud storage

**How cryptography is used**

- Full-disk encryption
- File-level encryption
- Secure key management

**What it protects**

- Data if devices are stolen
- Information during cloud breaches

**Real-world examples**

- Microsoft (BitLocker)
- Apple (FileVault)

---

## 7. Software and Update Verification

**Where it is used**

- Operating system updates
- Application installation
- Firmware upgrades

**How cryptography is used**

- Digital signatures verify the source
- Hash functions ensure integrity

**What it protects**

- Malware injection
- Fake or modified software updates

**Real-world example**

- Microsoft Windows
